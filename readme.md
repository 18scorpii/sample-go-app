# Execution Commands 

## To test

- cd ~/sample-go-app
- go test 

## To Run

- cd ~/sample-go-app
- go run main.go
 
## To Install

- cd ~/sample-go-app
- go install
- find the mac binary under HOME/bin

## To Compile

- `env GOOS=linux GOARCH=amd64 go build github.com/18scorpii/sample-go-app`
- Take the binary and run it on centos
